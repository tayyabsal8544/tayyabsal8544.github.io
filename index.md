## 🛠 Using the Dashboard

The dashboard serves as the control center for your automation. You do not need to interact with the game window directly. Every setting lives inside your web browser.

- Status Tab: This screen shows if the macro runs or pauses. It lists the current item count and your active trade orders.
- Settings Tab: Choose which items the macro tracks. You set your profit margins here. A higher percentage prevents the macro from taking low-profit trades.
- Logs: View the history of every transaction. The log shows buy times, sell times, and total coins earned during the session.
- API Key: You provide your Hypixel API key here. The program needs this to read bazaar prices from the game server.

## 🔑 Defining API Access

The application requires your unique Hypixel API key. This key identifies your account to the game servers. You generate this key inside the game.

1. Log into your Hypixel Skyblock profile.
2. Type /api new in the text box.
3. Copy the long string of letters and numbers that appears.
4. Paste this code into the API field in your dashboard.
5. Save the settings.

- Do not run multiple instances of the macro at once.

- Use a single account with this tool.
- Monitor your trades once every few hours.

## 🧩 Troubleshooting Common Issues

If the program fails to start or shows errors, check these common items.

- Firewall Access: Windows might block the program from reaching the internet. Check your Windows Defender settings and allow the application to bypass the blocks.
- Java Version: If nothing happens when you click the file, update your Java version to 17. The program cannot read the code without this specific environment.
- Connection Errors: Restart your internet router if the dashboard fails to load. Constant price updates require a steady link to the Minecraft servers.
- Empty Lists: If your dashboard displays no data, check the API key field. Re-generate a new key in the game if the old one expires.

## 💡 Best Practices for Flipping

The tool handles the work, but your strategy dictates the profit. Use these tips to maximize your return.

- Diversify Items: Do not put all your coins into one item. Split your budget across five or more different goods.
- Market Trends: Check the price history before you set your buy orders. Avoid buying items that show a steady price drop over the last week.
- Volume Matters: Focus on items with high daily volume. These items sell and buy within minutes. Low volume items trap your coins for long periods.
- Re-listing: Let the tool handle the auto-relisting feature. This keeps your orders at the top of the price list. It increases your chance to fill orders before other players.
